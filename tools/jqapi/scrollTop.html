返回值:IntegerscrollTop([val])

概述

获取匹配元素相对滚动条顶部的偏移。

此方法对可见和隐藏元素均有效。

参数

valString, NumberV1.2.6

设定垂直滚动条值

示例

无参数描述:

获取第一段相对滚动条顶部的偏移

HTML 代码:
<p>Hello</p><p>2nd Paragraph</p>
jQuery 代码:
var p = $("p:first");
$("p:last").text( "scrollTop:" + p.scrollTop() );
结果:
<p>Hello</p><p>scrollTop: 0</p>

参数val 描述:

设置相对滚动条顶部的偏移

jQuery 代码:
$("div.demo").scrollTop(300);
